# ztj1993/pptpd

这是一个 pptpd 镜像，用于快速搭建 pptp 服务器，并且支持在线配置用户名和密码。

## 项目链接
- [Github](https://github.com/ztj1993/docker-pptpd)
- [DockerHub](https://hub.docker.com/r/ztj1993/webproc)

## 快速应用
- 参考：docker-compose.yml
```
docker-compose up
```
